# Sileom Simulate

Sileom Simulate brings high-performance, GPU-accelerated, meshless static structural analysis directly into your nTop workflow. No external mesh preparation required. The solver works natively on nTop implicit geometry.

## Installation

Run the provided installer, which handles licensing automatically:

```
sileom-simulate-installer-2026-r2.msi
```

The installer sets up the connector interface and executable. No additional configuration is needed after installation.
