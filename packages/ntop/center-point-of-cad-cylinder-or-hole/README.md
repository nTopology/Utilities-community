# Center Point of CAD Cylinder or Hole

The CB will give you the Center Point of a CAD cylinder or hole.

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities.git
```

The package lives at `packages/ntop/center-point-of-cad-cylinder-or-hole/`.

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| `CAD Edge(s)` | list<brep_edge> | CAD Edges you wish to get the center point of. |

## Outputs

| Name | Type | Description |
| --- | --- | --- |
| `Centroid` | Point | Extract the centroid of the bounding box |
