# Mirror Points using a Plane

Mirror a set of points using a plane

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities.git
```

The package lives at `packages/ntop/mirror-points-using-a-plane/`.

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| `Point list` | Point List | The points you want to mirror |
| `Mirror plane` | plane | The plane that the points will mirror across |

## Outputs

| Name | Type | Description |
| --- | --- | --- |
| `Any_1` | Point List | Translates the projected points to the opposite side of the mirrored plane. |
