# Bounding Box for Points

This custom block creates a Bounding Box for a set of points

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities.git
```

The package lives at `packages/ntop/bounding-box-for-points/`.

## Inputs

| Name | Type | Description |
| --- | --- | --- |
| `Points` | Point List | The Points you want to create a Bounding Box for |

## Outputs

| Name | Type | Description |
| --- | --- | --- |
| `Bounding Box_0` | bounding_box | Creates a Bounding Box for a set of points based on the maximum and minimum coordinates of the point list |
