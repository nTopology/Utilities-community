# 2 Fluid Heat Exchanger

This generation of high-efficiency, dual-fluid heat exchangers is driven by Gyroid TPMS to maximize internal surface area and optimize thermal performance.

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities.git
```

The package lives at `packages/ntop/2-fluid-hex/`.

## Usage

1. Open the provided `.ntop` file in nTop 5.44.
