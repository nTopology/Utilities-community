# Point Index Field From Graph

Create a scalar field of the index values of nearest vertices in a graph.

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities.git
```

The package lives at `packages/DaveMakesStuff/point-index-field-from-graph/`.

## Usage

1. Open the provided `.ntop` file in nTop 4.12+.
2. Input graph to use to generate point index field

## Inputs & outputs

| Name | Type | Notes |
|------|------|-------|
| `Graph` | Graph | Graph to use to generate point index field |
| `Point Index Field` | Scalar Field | Main result |
