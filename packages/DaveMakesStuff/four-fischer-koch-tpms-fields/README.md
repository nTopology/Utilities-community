# Four Fischer Koch TPMS Fields

A set of four TPMS fields discovered by Werner Fischer and Elke Koch

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities-community.git
```

The package lives at `packages/DaveMakesStuff/four-fischer-koch-tpms-fields/`.

## Usage

1. Open the provided `.ntop` file in nTop 4.12+.
2. Input desired cell size
3. Select which Fischer-Koch field to generate (CS, CY, S, or Y)

## Inputs & outputs

| Name | Type | Notes |
|------|------|-------|
| `Cell Size` | Scalar value | Determines cell size of TPMS |
| `Fischer Koch` | Choice | Select which Fischer-Koch field to generate (CS, CY, S, or Y) |
| `Output` | Scalar Field | Output scalar TPMS field |
