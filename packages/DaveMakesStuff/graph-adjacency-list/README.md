# Graph Adjacency List

Creates an indexed text list of adjacent vertices for each vertex within a graph.

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities.git
```

The package lives at `packages/DaveMakesStuff/graph-adjacency-list/`.

## Usage

1. Open the provided `.ntop` file in nTop 4.12+.
2. Input graph to use

## Inputs & outputs

| Name | Type | Notes |
|------|------|-------|
| `Graph` | Graph | Graph to use to generate adjacency list |
| `Adjacency List` | Text List | Main result |
