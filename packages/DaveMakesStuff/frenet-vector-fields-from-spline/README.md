# Frenet Vector Fields from Spline

Create a vector field from the tangent, normal or binormal vectors of a spline.

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities-community.git
```

The package lives at `packages/DaveMakesStuff/frenet-vector-fields-from-spline/`.

## Usage

1. Open the provided `.ntop` file in nTop 4.12+.
2. Input spline
3. Select which vector field to generate (tangent, normal or binormal)

## Inputs & outputs

| Name | Type | Notes |
|------|------|-------|
| `Spline` | Spline | Input spline |
| `Field` | Choice| Select which vector field to generate (tangent, normal or binormal) |
| `Vector Field Output` | Vector Field | Output |
