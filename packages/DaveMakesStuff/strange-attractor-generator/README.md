# Strange Attractor Generator

Select from fifty-two "Strange Attractor" vector fields to generate visually interesting and mathematically intriguing curves.

[Watch the Tutorial Video]( https://youtu.be/KwzIOCl7qi0)

## Installation

Clone this repo or copy the package folder into your nTop workspace:

```bash
git clone https://github.com/nTopology/Utilities-community.git
```

The package lives at `packages/DaveMakesStuff/strange-attractor-generator/`.

## Usage

1. Open the provided `.ntop` file in nTop 4.12+.
2. Select a vector field and drag and drop to the "Selected Field" variable
3. Modify the field input parameters and input points to create interesting "strange attractor" designs.
4. Thicken the curves and export as is or export the vector field blocks to include in your own notebooks and custom blocks.
5. Refer to tutorial video for more details and demos.

## Inputs & outputs

| Name | Type | Notes |
|------|------|-------|
| Input Parameters | Scalar Values | Required. |
| Strange Attractor | Implicit Body | Main result. |
